# Análisis de Ventas de Videojuegos 🎮

Este proyecto analiza datos de ventas de videojuegos hasta 2016 para identificar patrones de éxito y ayudar a planificar campañas publicitarias en 2017.

## Contenido:
- Limpieza de datos
- Análisis por plataforma, género y región
- Visualización de datos
- Pruebas de hipótesis estadísticas

## Archivos:
- `analisis_videojuegos.ipynb` – Notebook principal con el análisis
